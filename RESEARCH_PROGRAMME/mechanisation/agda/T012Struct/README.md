### (A) The `ŌŗĆ`-aggregation lifts verbatim

A structured witness `SW` carries an **ASPIC+ attack type** `AType`
(`rebut` ŌĆö defeasible conclusion; `undercut` ŌĆö defeasible rule; `undermine`
ŌĆö ordinary premise; per [`lib/aspic/attacks.ts`](../../../../lib/aspic/attacks.ts),
Modgil & Prakken 2013) and a **read-polarity** `Pol` (`P`/`O`). The
structured Reading-C development is

```agda

```

i.e. T012's `ReadingC` instantiated at the structured witness type. So
`fidelity` (clause 1), `nesting-invariant` (clause 2), the permutation
`shift-neutral`, `conservativity`, `Accept-ŌåŁ`, `Accept-drop` (clause 4)
hold for structured witnesses **with no new proof** ŌĆö the aggregation is
transparent to argument structure. That is the "lift past abstract AF" for
the structure-agnostic clauses, made literal by module reuse.

### (B) Full mid-proof polarity re-typing

`retype1 : SW ŌåÆ SW` flips a witness's read-polarity ŌĆö an **involution**
(`retype1-invol`, via `flipPol-invol`). `retypeSched : List Bool ŌåÆ W ŌåÆ W`
re-types an arbitrary per-witness subset (a genuine mid-proof re-typing:
some witnesses flip, some do not). The new theorem:

```agda
retype-neutral : ŌłĆ Žü bs (w : W) ŌåÆ RC Žü w Ōćö RC Žü (retypeSched bs w)
```

Reading C is invariant under **any** re-typing schedule ŌĆö **strictly
stronger** than T012's permutation-only `shift-neutral` (it changes
polarities, not just order). `retype-then` composes re-typing with any
further verdict-equivalence (e.g. the permutation shift), so the full
clause-3 symmetry group is generated by **reorder Ōłś re-type**.

`retype-neutral` rides on one parameter:

```agda
conv-pol-sym : ŌłĆ Žü w ŌåÆ convS Žü w Ōćö convS Žü (retype1 w)
```

ŌĆö the per-pair verdict is symmetric under a read-polarity reflection. This
is the inherited **Ludics cut/orthogonality symmetry** (interaction is
symmetric in the two designs). Abstract AF could hide it because Dung edges
are polarity- and type-free.

### (C) The parameter is non-vacuous (honest scope)

`module Models` shows both sides:

- **Symmetric fragment** (`convSym`, polarity-blind ŌĆö the `rebut` case):
  `conv-pol-sym` holds **structurally** (`conv-pol-sym-holds = Ōćö-refl`), so
  the structured clauses + `retype-neutral` are inhabited unconditionally
  (`ex-RC`, `ex-retyped`).
- **Asymmetric fragment** (`convAsym`, polarity-**sensitive** ŌĆö an
  `undercut` that converges only when read with polarity `P`):
  `conv-pol-sym-fails` exhibits a concrete `w` with `convAsym Žü w` but not
  `convAsym Žü (retype1 w)`, so `┬¼ (conv-pol-sym)`. Full re-typing is
  therefore **not automatic** for the asymmetric attack types
  (undercut/undermine) ŌĆö it genuinely rides on the substrate cut-symmetry.

This is exactly the content abstract AF could not see: re-typing is free for
the symmetric (rebut / type-free) case, and load-bearing for the asymmetric
ASPIC+ attacks.
